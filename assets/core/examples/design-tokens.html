Token Usage Guidelines

When to Use SASS Variables

  • Typography: Font sizes, line heights, font weights
  • Spacing: Margins, padding, gaps
  • Border Radius: Corner rounding
  • Breakpoints: Responsive design breakpoints
  • Static Values: Values that don't change between themes

When to Use CSS Custom Properties

  • Colors: Text, background, border colors
  • Theme-dependent Values: Values that change between light/dark themes
  • Dynamic Values: Values that might be changed via JavaScript
  • User Customization: Values users might want to override

Token Naming Convention

// SASS Variables (Static)
$tutor-{category}-{variant}
$tutor-font-size-h1
$tutor-spacing-4
$tutor-radius-md

// CSS Custom Properties (Dynamic)
--tutor-{semantic-name}
--tutor-text-primary
--tutor-surface-l1
--tutor-border-idle
