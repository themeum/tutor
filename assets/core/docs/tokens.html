Tokens & SCSS

19 token files, light/dark themes, 11 mixin files, 32 component stylesheets, and utility class generators.

Build Pipeline

scss/main.scss
  ├── @use 'tokens'       → All design token variables & maps
  ├── @use 'themes'       → Light + Dark CSS custom properties
  ├── @use 'mixins'       → Reusable mixins (RTL, buttons, layout…)
  ├── @use 'components'   → 32 component stylesheets
  └── @use 'utilities'    → Utility classes from token maps

SASS Variables vs CSS Custom Properties

The token system uses a hybrid approach:

UseTypeReason
Typography, Spacing, Radius, ShadowsSASS variablesStatic — don't change between themes
Colors, Surfaces, Borders, Actions, TextCSS custom propertiesDynamic — change between light/dark
 
// ✅ Static values → SASS variables
padding: $tutor-spacing-6;          // 16px
font-size: $tutor-font-size-p1;     // 1rem
border-radius: $tutor-radius-lg;    // 8px

// ✅ Theme-aware → CSS custom properties
color: $tutor-text-primary;         // var(--tutor-text-primary)
background: $tutor-surface-l1;      // var(--tutor-surface-l1)
border-color: $tutor-border-idle;   // var(--tutor-border-idle)

Theme System

Themes set CSS custom properties on [data-tutor-theme="light"] and [data-tutor-theme="dark"]. All semantic tokens resolve to different primitive values per theme.

// themes/_light.scss
[data-tutor-theme='light'] {
  --tutor-text-primary: #{$tutor-gray-950};
  --tutor-surface-base: #{$tutor-gray-50};
  --tutor-surface-l1: #{$tutor-gray-1};
  --tutor-border-idle: #{$tutor-gray-200};
  --tutor-actions-brand-primary: #{$tutor-brand-600};
}

// themes/_dark.scss
[data-tutor-theme='dark'] {
  --tutor-text-primary: #{$tutor-gray-50};
  --tutor-surface-base: #{$tutor-gray-950};
  --tutor-surface-l1: #{$tutor-gray-900};
  --tutor-border-idle: #{$tutor-gray-700};
  --tutor-actions-brand-primary: #{$tutor-brand-500};
}

Mixins 11 files

RTL Mixins

All directional properties should use RTL mixins for automatic dir="rtl" support:

@include margin-start($tutor-spacing-4);   // margin-left → margin-right in RTL
@include margin-end($tutor-spacing-4);     // margin-right → margin-left in RTL
@include padding-start($tutor-spacing-4);  // padding-left → padding-right in RTL
@include border-start(1px solid red);      // border-left → border-right in RTL
@include text-align-start;                 // text-align: left → right in RTL
@include inset-inline-start(0);            // left: 0 → right: 0 in RTL
